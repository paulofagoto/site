O que é o Ventoy?

Uma ferramenta revolucionária para criar pendrives USB bootáveis com múltiplos sistemas operacionais. Basta instalar o Ventoy no pendrive uma única vez e depois copiar os arquivos ISO desejados. Sem complicações, sem regravações constantes.

Multiboot Instantâneo

Copie quantas ISOs quiser para o pendrive. O Ventoy exibe um menu de boot para você escolher qual sistema iniciar. Suporte a Windows, Linux, BSD, Unix, WinPE e muito mais.

BIOS e UEFI

Funciona perfeitamente tanto em computadores antigos com Legacy BIOS quanto nos modernos com UEFI (incluindo Secure Boot). Instalação rápida e compatível com GPT e MBR.

Persistência

Suporte a arquivos de persistência para sistemas Linux, permitindo salvar alterações e configurações diretamente no pendrive, como se fosse um sistema instalado.

Instalação Única

Instale o Ventoy no pendrive uma vez e depois apenas copie novas ISOs quando quiser. Não precisa formatar ou regravar o pendrive a cada novo sistema.

Como usar o Ventoy

Em poucos passos você terá um pendrive USB capaz de iniciar vários sistemas operacionais.

  1. Baixe o Ventoy

    Clique no botão verde e baixe o arquivo mais recente do SourceForge. O download inclui o executável para Windows ou o pacote para Linux.

  2. Execute o Ventoy2Disk

    No Windows, execute o Ventoy2Disk.exe. Selecione o pendrive USB desejado (atenção: todos os dados serão apagados) e clique em Install.

  3. Copie as ISOs

    Após a instalação, o pendrive aparecerá como uma unidade normal no Windows Explorer. Basta copiar e colar os arquivos .iso dos sistemas operacionais que deseja.

  4. Configure o Boot

    Reinicie o computador e acesse o menu de boot (F12, F8, Esc, etc.). Selecione o pendrive USB. O menu gráfico do Ventoy aparecerá listando todas as ISOs copiadas.

  5. Inicie qualquer sistema

    Use as setas do teclado para escolher a ISO desejada e pressione Enter. O sistema será iniciado normalmente, como se tivesse sido gravado individualmente.

  6. Atualize quando quiser

    Para adicionar novas ISOs, basta copiá-las para o pendrive. Para atualizar o Ventoy, execute novamente o instalador e selecione a opção Update (seus arquivos serão preservados).

Dica de ouro: Mantenha sempre um pendrive com Ventoy pronto em sua mochila ou chaveiro. Com várias ISOs de recuperação e instalação, você estará preparado para qualquer emergência técnica.
