O que é o 3DP Net?

Um programa leve, portátil e gratuito que contém uma vasta coleção de drivers de rede (ethernet e Wi-Fi) para Windows. Ideal para quando você formata o PC e perde a conexão com a internet.

Instalação Offline

Não precisa de internet para funcionar. Basta ter o programa salvo em um pendrive e executá-lo no computador formatado.

Detecção Automática

O 3DP Net identifica automaticamente o modelo da sua placa de rede e seleciona o driver correto para instalação.

Drivers Integrados

Todos os drivers já vêm embutidos no programa. Suporta a grande maioria das placas de rede PCI, PCIe, onboard e USB.

Windows XP, Vista e 7

Compatível com Windows 2000, XP (32/64 bits), Vista (32/64 bits), Windows 7 (32/64 bits), Server 2003 e 2008.

Ethernet e Wi-Fi

Funciona tanto para placas de rede cabeadas (ethernet) quanto para adaptadores sem fio (wireless).

Totalmente Grátis

Programa 100% gratuito, sem custos escondidos. Desenvolvido pela 3DP, referência em instalação de drivers.

Como usar o 3DP Net

Siga estes passos simples para recuperar o driver de rede do seu computador após formatar o Windows.

  1. Baixe o 3DP Net em outro computador

    Use um PC que tenha internet para baixar o arquivo do 3DP Net. Salve-o em um pendrive.

  2. Copie para o computador sem internet

    Conecte o pendrive no computador que está sem driver de rede e copie o arquivo para a área de trabalho.

  3. Execute o programa

    Clique duas vezes no arquivo 3DP_Net.exe. O programa abrirá e fará uma varredura automática da sua placa de rede.

  4. Instale o driver

    Após a detecção, clique no botão de Instalar Driver. O 3DP Net instalará o driver correto para sua placa de rede.

  5. Reinicie o computador (se necessário)

    Em alguns casos, é necessário reiniciar o Windows para que o driver entre em funcionamento. Após reiniciar, a internet estará funcionando!

  6. Use o 3DP Chip para os demais drivers

    Com a internet funcionando, utilize o 3DP Chip para baixar e instalar automaticamente todos os outros drivers do seu computador (vídeo, áudio, chipset, etc.).

Dica de ouro: Mantenha sempre uma cópia do 3DP Net em um pendrive de emergência. Assim, sempre que formatar um computador, você terá o driver de rede à mão, sem depender de outro PC para baixar.
